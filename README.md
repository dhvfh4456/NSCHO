## 项目分支结构

我们的项目采用多分支开发和维护模式，以适应不同功能和版本需求。分支结构如下：

- **main**  
  主分支，稳定版本发布与日常维护。

- **dev**  
  开发分支，所有新功能和优化均在此分支集成测试后再合并至主分支。

- **feature/wiki**  
  Wiki相关功能开发分支，专注于Wiki界面及资料完善。

- **feature/server**  
  动态服务器功能开发分支，持续迭代中。

- **hotfix**  
  紧急修复分支，用于快速修复线上发现的问题。

- **release**  
  版本发布分支，集成并测试各功能，准备最终发布。
